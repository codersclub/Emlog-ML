Introduction

  • emlog is a short for "Every Memory Log", this means: bit the memory. It is a powerful blog and CMS website system Based on PHP and MySQL. It is dedicated to provide a fast and stable, and extremely simple in usage, comfortable service for create a content for your site.

1 minute installation

  • 1.The src folder of all the files uploaded to the server you want to install emlog directory.
  • 2. Visit your site domain name in the browser installation interface will appear, according to the prompts to fill submitted.
  • 3. After a successful installation, start your creative bar.
